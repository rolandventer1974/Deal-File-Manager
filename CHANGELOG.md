# Changelog

All notable changes to the Deal File Manager project will be documented in this file.

## [1.0.0] - 2026-03-23

### Added
- Initial release of Deal File Manager
- Dashboard with deal file listing and status
- Deal file creation form with customer and vehicle details
- Document upload functionality with validation
- REST API for ColdFusion integration
- OTP (Offer to Purchase) submission via API
- Document management and tracking
- Activity logging for all operations
- Responsive web interface using Bootstrap
- Database schema with complete data models
- Nginx and Apache configuration files
- Automated installation script for Digital Ocean
- API integration guide for ColdFusion developers
- Comprehensive documentation and deployment guide
- File manager with permission controls
- API response utilities
- Authentication and security utilities
- Logger with file and database logging

### Features
- Multi-format document support (PDF, Word, Images, Spreadsheets)
- Deal file status tracking and completion percentage
- Document status management (pending, verified, rejected)
- Sorting and filtering of deal files
- Pagination support
- RESTful API with Bearer token authentication
- Automatic database backups
- SSL/HTTPS support
- Mobile-responsive interface

### Security
- SQL injection prevention with prepared statements
- File type validation
- File size restrictions (50MB max)
- API token authentication
- Input validation and sanitization
- CORS configuration

### Documentation
- Complete README with installation instructions
- Digital Ocean deployment guide
- ColdFusion API integration guide
- Installation scripts for automated setup
- API documentation with examples
